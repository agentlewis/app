# holodex app

an app for how we relate to people and groups within a network. see [top-level repo](https://github.com/open-app/holodex) for more info.

## how to

### install

```bash
mkdir -p ~/repos/holodex
cd ~/repos/holodex
git clone https://github.com/holodex/app
cd app
npm install
```

### develop

```
npm run develop
```

### start

```
npm run start
```

### deploy

```
npm run deploy
```

### test

```
npm run test
```
